Question-1: What are the different ways to select an element in the DOM?

To select an element in DOM we generally use : document.getElementById('idName'), document.getElementsByClassName('className'), document.getElementsByTagName('tagName'), document.querySelector(".className/#idName"), document.querySelectorAll(".className/#idName").

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: With innerHTML, we can select all the HTML elements with html tags, text, spacing inside the selected element
innerText: With innerText, we can select all the text content as output but can't choose HTML tags or CSS style.
textContent: Same as innerText. We can get text content as output and also get hidden items.

Question-3: What is event delegation in the DOM?

With the help of event delegation, we can handle event easily by using event bubbles. It helps to manage event more dynamically using child & parent element.

Question-4: What is event bubbling in the DOM?

Event bubbling means that when an event occurs on an element, it propagates up through its parent elements in the DOM. For example, if you select an element in HTML, you can access its parent element, and subsequently, the parent's parent element, and so on. This process continues like a bubble rising to the surface, allowing parent elements to respond to events that occurred on their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add and remove elements, we usually use these method:
Create elements: document.createElement('TagName');
Add element: parentElement.appendChild('createdElement');
Remove elements: parentElement.removeChild('elementName');
